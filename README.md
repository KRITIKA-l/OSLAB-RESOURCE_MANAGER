# Smart OS Resource Manager

Smart OS Resource Manager provides interactive simulations of key operating-system algorithms.

## Features

- CPU Scheduling Algorithms
	- FCFS, SJF, SRTF, Priority (Non-Preemptive), Priority (Preemptive), Round Robin
- Memory Management Algorithms
	- FIFO and LRU page replacement
- Deadlock Avoidance
	- Banker's algorithm with safe-state and safe-sequence evaluation
- Rich Output Metrics
	- Completion Time (CT), Turnaround Time (TAT), Waiting Time (WT)
	- Average WT and average TAT
	- Page fault count, need matrix, and timeline visualization
